- 型指定
  - nullを許容する場合
    - `str string | null`
- interface
  - 関数に使う場合：引数と返り値の型指定
    - Reactコンポーネントではそれをpropsと呼ぶ
  - クラスに使う場合：フィールドの型指定とメソッドの型と返り値指定
  - ?でオプショナルなものも定義できる
  - interfaceも継承できる
